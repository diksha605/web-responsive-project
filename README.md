# 📢 Digital Marketing Agency Website

A fully responsive **Digital Marketing Agency** website built with **HTML**, **CSS**, **Bootstrap 5**, and **JavaScript**. This project showcases a modern, professional web presence for a digital marketing business — including services, portfolio, testimonials, and a contact section.

---

## ✨ Features

- ✅ Fully **responsive design** using Bootstrap 5 grid system
- ✅ **Sticky navigation bar** with smooth scroll
- ✅ **Hero section** with call-to-action buttons
- ✅ **Services section** with icon cards
- ✅ **Portfolio / Work showcase** section
- ✅ **Client testimonials** carousel
- ✅ **About Us** section
- ✅ **Contact form** with validation
- ✅ **Footer** with social media links
- ✅ Mobile-first, cross-browser compatible


2. **Navigate into the project folder**

## 📱 Responsive Breakpoints

This project follows Bootstrap's standard responsive breakpoints:

| Breakpoint | Screen Size |
|---|---|
| Extra small (xs) | < 576px |
| Small (sm) | ≥ 576px |
| Medium (md) | ≥ 768px |
| Large (lg) | ≥ 992px |
| Extra large (xl) | ≥ 1200px |
